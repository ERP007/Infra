# Harbor Secrets

Place the trusted TLS certificate and private key for `registry.erp007.xyz` here
on the server before running `scripts/setup-harbor.sh`.

Expected server-only files:

```text
server-secrets/harbor/registry.erp007.xyz.crt
server-secrets/harbor/registry.erp007.xyz.key
server-secrets/harbor/admin-password
server-secrets/harbor/database-password
```

`admin-password` and `database-password` are generated by the setup script if
they do not exist.
